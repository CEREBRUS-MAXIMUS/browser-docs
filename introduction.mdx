### Root Endpoint

To verify that the server is up and running.

**Example Request:**

```http
GET http://localhost:1234/
```

**Example Response:**

```text

```

## Session Management

### Start Session

#### `POST /startSession`

**Purpose:** 
To start a new session and return a unique session ID.

**Parameters:**
- `url` (optional): URL to open in the session. Defaults to 'https://www.google.com' if not provided.

**Example Request:**

```http
POST http://localhost:1234/startSession
Content-Type: application/json

{
  "url": "https://www.example.com"
}
```

**Example Response:**

```json
{
  "sessionId": "unique-session-id"
}
```

### End Session

#### `POST /endSession`

**Purpose:** 
To end a session and close the associated browser window.

**Parameters:**
- `sessionId` (required): ID of the session to end.

**Example Request:**

```http
POST http://localhost:1234/endSession
Content-Type: application/json

{
  "sessionId": "unique-session-id"
}
```

**Example Response:**

```json
{
  "success": true,
  "message": "Session ended successfully"
}
```

### Get Current Sessions

#### `GET /currentSessions`

**Purpose:** 
To retrieve all current sessions.

**Example Request:**

```http
GET http://localhost:1234/currentSessions
```

**Example Response:**

```json
[
  {

    "currentMarkdown": "",
    "currentDOMRepresentation": {},
    "currentTools": [],
    "currentURL": "https://www.example.com",

```
